# University Beginner Programs 🎓

This repository contains beginner-level C programs and university coursework exercises based on my first-semester programming syllabus.

## Topics Covered

- Input and output
- Variables and data types
- Operators and expressions
- Conditional statements
- Loops
- Functions
- Arrays
- Strings
- Structures
- Pointers
- Dynamic memory allocation using `malloc()` and `calloc()`
- File handling
- Basic problem solving exercises

## Purpose

The purpose of this repository is to build a strong foundation in C programming and problem solving during my first year at university.
